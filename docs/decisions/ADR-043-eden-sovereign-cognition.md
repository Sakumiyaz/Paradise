# ADR-043: Eden Sovereign Cognition

## Status

Accepted

## Context

OpenCog Hyperon is strongest where it has a mature formal knowledge substrate, MeTTa language, symbolic reasoning lineage, distributed AtomSpace direction and external ecosystem history. EDEN should not copy AtomSpace, MeTTa or metagraph rewriting as its core answer.

EDEN's stronger route is original: governed cognition in action. The system should define superiority targets around GEWC authority, Praxis Nexus, evidence-bound memory, Cognitive Contract Language, policy-safe reasoning, governed distribution and reproducible maturity.

## Decision

Add `sovereign cognition eval` as the validation command for Eden Sovereign Cognition.

It formalizes 11 local architecture win sectors against OpenCog Hyperon:

- central core
- mathematical formalism
- distributed scalability concept
- design originality
- memory and knowledge
- multi-paradigm integration
- external ecosystem
- historical maturity
- symbolic reasoning
- cognitive language
- formal representation

It writes:

- `eden_sovereign_cognition.json`
- `sovereign_sector_wins.json`
- `praxis_calculus_formalism.json`
- `cognitive_contract_language.json`
- `evidence_memory_fabric.json`
- `federated_runtime_fabric.json`
- `symbolic_reasoning_fabric.json`

The external ecosystem sector depends on ADR-044 and the `external ecosystem eval` artifacts rather than treating SDK/adapters alone as enough evidence.

The design remains no-claim. The command asserts local architecture evidence and superiority targets, not AGI capability, market adoption, elapsed historical age or independent external validation.

## Consequences

- EDEN now has an explicit original path to surpass Hyperon sectors without copying Hyperon primitives.
- Praxis Nexus is deepened into Praxis Calculus, Cognitive Contract Language, evidence-bound memory, governed runtime federation and symbolic reasoning fabric.
- Release-candidate packaging, local held-out validation, capability registry and independent package validation now require sovereign cognition artifacts.
- The external ecosystem and historical maturity sectors are framed as auditable maturity targets, not unsupported claims that EDEN already has more years or broader community than OpenCog.
