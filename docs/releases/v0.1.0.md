# Paradise v0.1.0

Date: 2026-05-25

Status: first public release. This is not an AGI capability claim and not a
trained model release.

## Scope

Paradise v0.1.0 publishes the local-first Worldcell Runtime surface:

- GARM/GEWC operator runtime;
- Paradise Worldcell session loop;
- runtime spine contracts and enforcement reports;
- operational, runtime-state, GEWC, Paradise and action-contract APIs;
- `edenctl` operator CLI and `eden_core::sdk` Rust client;
- `contracts/v1` schema, OpenAPI and example package;
- static operator console;
- local conformance, black-box, smoke, long-run and public-audit gates.

## Public Boundary

Paradise is the public product and repository identity. Eden remains the broader
architecture substrate represented inside the workspace. GARM and GEWC remain
native implementation terms for the governed runtime body and executive core.

The release does not claim:

- completed AGI;
- trained LLM/LMM capability;
- autonomous deployment readiness;
- benchmark superiority;
- production safety certification.
