# Paradise Examples

These examples are copyable operator workflows for the public Paradise surface.
They intentionally start with dry-run and evidence generation instead of model
keys, external tools or autonomous writes.

| Example | Purpose |
| --- | --- |
| [repo-inspection](repo-inspection/README.md) | Plan a repository inspection without touching files. |
| [safe-file-action](safe-file-action/README.md) | Show how a file-organizing action should be represented before approval. |
| [worldcell-session](worldcell-session/README.md) | Generate Worldcell evidence and a dry-run session together. |
